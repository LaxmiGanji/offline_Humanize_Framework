# 🧠 Offline Humanized Text Summarizer

## 📋 Overview
An advanced AI-powered text summarizer that generates **human-like, plagiarism-free summaries** completely offline. Perfect for students, researchers, and faculty who need unique summaries without internet dependency.

## ✨ Key Features
- **🤖 Multiple AI Models**: T5-Small (fast), T5-Base (balanced), BART-Base (best quality)
- **📴 100% Offline**: After first download, works without internet
- **🔄 Plagiarism-Free**: Generates new sentences, never copies
- **🎨 Humanized Output**: Natural language with proper flow and transitions
- **📊 Multiple Variations**: Get 3 different summaries to choose from
- **📂 File Support**: Load .txt files and save summaries
- **🎛️ Customizable**: Control length and creativity level
- **📈 Word Counter**: Real-time word and character count
